### 1.类的组合是一个大的类里面包含很多小的对象
### 2.组合类构造函数定义的一般形式为：
类名：：类名（形参表）：内嵌对象1（形参表），内嵌对象2（形参表）···<br>
{<br>类的初始化<br>}<br>
复制构造例子：<br>
```C++
C::C(C &c1):b(c1.b)
{
   ...
}
```
当内嵌对象有无参构造函数时，初始化列表中的语句可省略。<br>
根据对象在类中的定义顺序调用构造函数。析构函数调用顺序相反，先调用组合再调用内嵌类。
